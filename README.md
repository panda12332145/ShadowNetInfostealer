# ShadowNetInfostealer
ShadowNet Infostealer é um malware Windows que coleta dados (credenciais, Wi-Fi, arquivos) e exfiltra via FTP. Inclui módulos de perturbação (páginas pornográficas, alertas, desligamento) para causar dano. Escrito em BAT/VBScript, sem dependências externas, compatível com Windows 7+.
